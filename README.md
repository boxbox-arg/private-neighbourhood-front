# Private Neighbourhood - Frontend

Aplicación web mobile-first para administración de barrios privados.

## Requisitos

- Node.js 18+
- **pnpm** (gestor de paquetes obligatorio)

```bash
corepack enable
corepack prepare pnpm@latest --activate
```

## Instalación y desarrollo

```bash
pnpm install
pnpm dev
```

## Scripts

| Comando   | Descripción           |
|-----------|-----------------------|
| `pnpm dev`     | Servidor de desarrollo |
| `pnpm build`   | Build de producción   |
| `pnpm preview` | Preview del build     |
| `pnpm lint`    | Linter                |
